# VZShop E-commerce Database Design & Sales Analysis Using SQL

## Project Overview

VZShop is a merchandise store for coders selling hoodies, mugs, stickers, notebooks, and accessories. This project focuses on designing a relational database system and performing SQL-based business analysis.

## Database Features

* Relational database design
* Primary and foreign key relationships
* Customer, product, order, payment, and inventory management

## SQL Analysis Performed

* Best-selling products
* Low-selling products
* Monthly revenue analysis
* Repeat customers
* Completed vs pending orders
* Product sales ranking
* Running revenue totals

## Tools Used

* SQL
* MySQL

## Skills Demonstrated

* Database Design
* Joins
* Aggregate Functions
* Window Functions
* Business Analysis
